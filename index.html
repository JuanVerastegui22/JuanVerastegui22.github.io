02

Prompts Used

C8: "Build an interactive HTML+CSS+JS webpage that lets users pick a probability distribution (uniform, triangular, linear, piecewise, normal), adjust parameters, and shade/calculate probabilities below, above, or between two points using Chart.js."

C9: "Extend the webpage with a data-modeling tab where users upload a CSV, select a column, choose a distribution, and see an empirical histogram overlaid with the fitted PDF, along with estimated parameters and a fit interpretation."

03

How AI Was Used

  • Generated the initial scaffold for the HTML/CSS/JS structure.
  • Provided the mathematical formulas for each PDF.
  • Wrote the Chart.js integration code for shaded probability regions.
  • Suggested parameter estimation methods (MLE for normal, min/max for uniform, histogram-based for piecewise).
  • Generated the CSV parsing and column-detection logic.
04

What We Verified & Corrected

  • Verified that all PDFs integrate to 1 over their support.
  • Tested piecewise area validation and added the warning message.
  • Corrected the triangular CDF formula for probabilities when x is above the mode.
  • Verified normal probability calculations against Z-table values.
  • Tested each built-in dataset and confirmed histograms render correctly.
05

Testing Process

  • Each distribution was tested with known parameter combinations and probabilities verified manually.
  • All three provided CSV datasets (Dataset 1, 2, 3) were loaded and fitted with all five distributions.
  • Edge cases tested: σ → 0 for normal, a = b for uniform, mode at endpoints for triangular.
  • Probability calculations cross-checked using standard normal tables for the normal distribution.
